First Put your photo in the folder images insert a folder containing image of a person with
the person name as the folder name.
Then run the face-train.py then there will be a file name tarinner.yml
Then run faces.py there'll be a pop up camera and will recognize your face, if the tag is wrong,
add more of your picture in the file and make sure that your picture is well lighted and
have many angle of your face.

*there should be more then 1 file in the images folder
