How:
1. Insert Your document into a folder and copy the folder path ex:
C:\Users\Asus\Documents\BINUS\NLP
2. Paste the folder Path (YOUR PATH) inside LDetection.py 
like this
Ex:
organize_files_by_language(r'C:\Users\Asus\Documents\BINUS\NLP\ProjectNLP')
3. Run the .py file
4. open the folder location
5. Tadaaa your docx is sorted
